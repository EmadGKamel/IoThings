# IoThings
## About
IoThings is MQTT client desktop application made using PyQT5 and paho-mqtt.
## Running 
```
$ git clone https://github.com/EmadGKamel/IoThings.git
$ cd IoThings/
$ python main.py
```
## dependencies
### paho-mqtt
```
sudo pip install paho-mqtt
```
### PyQT5
```
sudo pip install PyQt5
```
